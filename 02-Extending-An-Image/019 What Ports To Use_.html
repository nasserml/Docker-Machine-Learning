What Ports To Use?


Hi all!


We have seen ports mentioned a few times for Docker, and how we need to choose a port along with mapping the container port to it. There are some default ports, for example port 5000 with flask and we will also see port 80 used often. Standard HTTP and HTTPS ports would be 80 and 443 which might look familiar or might come up often.


For further information this article can be helpful:


https://www.stackrox.com/post/2017/08/hardening-docker-containers-and-hosts-against-vulnerabilities-a-security-toolkit/





When using docker -p it will help us create our port rules but we also have the expose command for the following:


EXPOSE <port> [<port>/<protocol>...]

The EXPOSE instruction informs Docker that the container listens on the specified network ports at runtime. You can specify whether the port listens on TCP or UDP, and the default is TCP if the protocol is not specified.


https://docs.docker.com/engine/reference/builder/


For a general rule do not or please be careful when mapping any ports below 1024 with a container because Docker does allow containers to be mapped to privileged ports. Please only ensure necessary ports are available for Docker containers.
